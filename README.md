# Quantum-Quotient-Analysis
Developed a Streamlit application using Nifty-50 stock market data to create various interactive statistical plots, including time series plots, candlestick charts, VWAP, MACD, RSI, and Bollinger Bands, enabling users to generate meaningful insights and make informed decisions.

# Here's a quick overview of the components in the project. 
1. Stock Market analytics table, sidebars
2. Loading the data
3. Additional information about the stock
4. Time series plots
	-Closing price over time
	-Volume traded over time
	-Opening vs closing prices over time
5. Time series decomposition 
6. OHLC chart - candlestick chart
7. Calculating and plotting VWAP over time
8. Number of trades over time - trades analysis
9. Comparative closing prices
10. Heatmap of correlation matrix 
11. Pair plot
12. Histogram of returns 
13. MACD indicator
14. Cumulative return plot 
15. Relative performance comparison
16. Annualized volatility
17. RSI chart
18. Bollinger bands
19. On-balance volume 
20. Simple moving average
21. Exponential moving average
22. Average true range
23. Stochastic oscillator
24. Chaikin money flow
25. Binning
26. Normalization
27. Sampling
28. Model for predicting close - Random forest 
29. Time series plots with animation - closing price over time, volume traded over time
30. 3-D Interactive plots - 3-D Scatter plot for high, low and close prices, 3-D Surface plot of close prices over time
